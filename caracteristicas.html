Características del HTML Moderno

Etiquetas semánticas

Estructuran el contenido con significado, mejorando accesibilidad y SEO.

  • header — Encabezado de página o sección
  • nav — Navegación principal
  • section — Sección temática
  • article — Contenido independiente
  • aside — Contenido complementario
  • footer — Pie de página o sección

Multimedia

Inserción nativa de audio, video y gráficos sin plugins.

  • video — Reproducción de vídeo
  • audio — Reproducción de audio
  • canvas — Gráficos 2D dinámicos
  • svg — Gráficos vectoriales
  • source — Múltiples fuentes de medios

Formularios modernos

Nuevos tipos de input y validación integrada.

  • email, url, tel
  • number, range, date, time
  • color, search
  • placeholder, required, pattern
  • datalist para sugerencias

APIs y funcionalidades avanzadas

Interfaces que permiten aplicaciones web ricas.

  • Geolocalización (Geolocation API)
  • Almacenamiento local (localStorage, sessionStorage)
  • Arrastrar y soltar (Drag and Drop)
  • Historial (History API)
  • Web Workers para tareas en segundo plano
