Historia del HTML

1991

HTML 1.0

Tim Berners-Lee creó el primer documento formal que describía HTML como un lenguaje de marcado. Contenía aproximadamente 18 etiquetas básicas para títulos, párrafos, listas y enlaces. Fue el origen de la World Wide Web en el CERN.

1995

HTML 2.0

Primera especificación oficial del HTML publicada por el IETF (Internet Engineering Task Force). Estableció un estándar común para que los navegadores interpretaran el mismo código de forma consistente. Incluyó formularios básicos y tablas.

1997

HTML 3.2

Desarrollado por el W3C. Añadió soporte para tablas completas, applets de Java, texto alrededor de imágenes y elementos de presentación. Fue una versión de transición que incorporaba muchas características que los navegadores ya habían implementado por su cuenta.

1999

HTML 4.01

Versión estable y muy utilizada. Introdujo hojas de estilo (CSS), scripts (JavaScript), marcos (frames) y una mejor separación entre estructura y presentación. Se publicaron las variantes Strict, Transitional y Frameset. Dominó la web durante más de una década.

2000

XHTML (versión XML)

Reformulación del HTML 4 como aplicación XML. Exigía sintaxis más estricta: etiquetas cerradas, atributos entre comillas y nombres en minúsculas. Buscaba compatibilidad con otras tecnologías XML y un código más limpio, aunque con el tiempo HTML5 retomó un enfoque más pragmático.

2014

HTML5

Recomendación oficial del W3C en octubre de 2014. Revolucionó la web con etiquetas semánticas (header, nav, section, article, footer), elementos multimedia nativos (video, audio, canvas), formularios mejorados con nuevos tipos de input, almacenamiento local y APIs para geolocalización, arrastrar y soltar, y mucho más. Es el estándar actual de la web.
